SEO tool

Have you ever wondered how Google search works? How exactly does it provide the websites that we are searching for?

Briefly expalined, when you type in a keyword in Google search box, Google crawls the internet across hundreds of thousands of pages(!) for the keyword that you have entered. The website which has the most number of occurrances of the website is displayed first and the other websites in descending order. This is the very basic working of Google or any other search engine for that matter.

Then what is SEO??

SEO stands for Search Engine Optimisation. It is to make sure that your website is the first link that is displayed when a user searches for the keyword relating to your website.

What is this project about?

The python script that I have written does a part of what Google's super secret algorithm does to find relevant websites and make users happy.

There are two python scripts that I have written.

The first one

The first one is to find out the number of occurrances of a word in your website which greatly helps Search Engines such as Google display your website at the top. This python script allows the user to enter the URL of the website and search for the number of occurrances of the word. It then displays the frequency of the word in that particular website.

When is this helpful?

For example, there is a link in a website which says 'Find more about this topic here!'. You are exhausted reading through several websites already and don't want to read stuff anymore. You just want to know whether that website has something to do with the damn topic you are searching for. In that case, fret not! Just copy paste the url link from the website and find out how many times the word has occurred in the website. And Voila! Your work will be cut in half and double your time will be saved to do more fun stuff!!!

Pros

  • Very user friendly (exceptional error handling)

Cons

  • Cannot search through multiple websites

The second one

This is very similar to the first one except that, this is more like Google. It searches through all the URLs that you enter and displays how many times it has occurred in a website. The one with most occurrances is probably the first place you should go and look for your answers.

When is this helpful?

Similar to the first one, but this time, it can handle multiple websites!!

Pros

  • Can handle multiple websites

Cons

  • Error handling isn't very user friendly (i.e. user has to strictly follow the instructions)
  • Speed at which the program runs depends on the user's computer/laptop
